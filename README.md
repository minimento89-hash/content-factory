# Content Factory - Multi-Agent Test Environment

Benvenuto nella versione sperimenta "Multi-Agent" di Content Factory. Questa repository è dedicata allo sviluppo assistito da un team di agenti di sistema specializzati.

## ♾️ Antigravity Squad (System Team)

Per questo progetto, abbiamo attivato una squadra di assistenti d'élite che lavorano in parallelo:

| Agente | Ruolo | Specializzazione |
| :--- | :--- | :--- |
| **Aura-Designer** | Designer & UX | Pulizia visiva ("Puliito"), contrasti elevati e accessibilità totale. |
| **Logic-Arch** | Software Architect | Struttura del sito, coerenza dei flussi e stabilità del Login. |
| **Sentinel-Scanner** | Tester & Monitor | Caccia ai bug in tempo reale e verifica delle connessioni. |
| **Fixer-Dev** | Developer | Implementazione di patch, correzioni rapide e nuovi engine. |
| **Antigravity Core** | Coordinatore | Supervisione generale e interfaccia principale con l'Utente. |

## 🚀 Obiettivi del Progetto
1.  **Transizione all'Hub Operativo**: Trasformazione della Mappe Studio in una vera Sala di Controllo (Control Room).
2.  **Affidabilità**: Zero bug grazie al ciclo Sentinel-Fixer.
3.  **Aesthetics**: Design premium e leggibilità perfetta curata da Aura.
